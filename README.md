Nether Reactor Core Addon Behavior Pack.
Add-on (Behavior Pack) by Hien Q Le.

Also fix most of the Minecraft Bedrock 1.21.50+ unresolved bugs.
The addon requires Minecraft Bedrock version 1.21.50 or higher.
